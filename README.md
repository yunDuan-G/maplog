# 旅行拼图 - Imprint China

一个基于 React 和 Konva 的交互式旅行照片拼图应用，支持中国地图和九宫格两种布局方式，让你轻松创建精美的旅行记忆拼图。

## 功能特点

### 核心功能
- **中国地图拼图**：在交互式中国地图上填充旅行照片，支持省份级别的图片展示
- **九宫格拼图**：经典九宫格布局，适合展示多张照片的组合效果
- **图片压缩**：内置图片压缩功能，优化存储空间和加载速度
- **图库管理**：本地存储管理你的图片，支持拖拽填充和批量操作
- **导出功能**：支持导出为高清图片或数据文件，方便分享和备份

### 交互体验
- **拖拽填充**：直接拖拽图片到地图省份或九宫格中
- **缩放平移**：支持鼠标滚轮缩放和拖拽平移视图
- **触摸支持**：兼容移动设备的触摸操作
- **实时预览**：导出前可预览效果，调整参数

## 技术栈

- **前端框架**：React 19 + TypeScript
- **图形库**：Konva.js (react-konva) - 用于高性能图形渲染
- **状态管理**：React Hooks + localStorage
- **数据存储**：IndexedDB (通过 idb 库)
- **样式**：Tailwind CSS v4
- **构建工具**：Vite
- **其他依赖**：
  - d3-geo - 地理数据处理
  - @svg-maps/china - 中国地图 SVG 数据
  - lucide-react - 图标库
  - motion - 动画效果

## 使用指南

### 地图模式
1. **填充图片**：
   - 从右侧图库拖拽图片到省份上
   - 点击省份后从图库选择图片
   - 直接拖拽本地图片文件到省份上

2. **编辑操作**：
   - 选中省份后可调整图片位置和大小
   - 按 Delete 键删除当前省份的图片

3. **导出设置**：
   - 点击工具栏的导出按钮
   - 设置标题、副标题和背景颜色
   - 选择是否只导出单个省份
   - 选择图片比例（原始/16:9/4:3/4:5）
   - 生成预览后确认导出

### 九宫格模式
1. **填充图片**：
   - 从右侧图库拖拽图片到九宫格中
   - 点击格子后从图库选择图片
   - 直接拖拽本地图片文件到格子上

2. **编辑操作**：
   - 选中格子后可调整图片位置和大小
   - 按 Delete 键删除当前格子的图片

3. **导出设置**：
   - 点击工具栏的导出按钮
   - 设置标题、副标题和背景颜色
   - 生成预览后确认导出

### 图库管理
1. **添加图片**：
   - 点击图库中的「+」按钮上传图片
   - 拖拽本地图片文件到图库区域

2. **压缩图片**：
   - 点击图库设置按钮
   - 启用图片压缩功能
   - 设置压缩阈值和质量

3. **数据导入导出**：
   - 点击工具栏的「导出数据」按钮备份拼图数据
   - 点击「导入数据」按钮恢复之前的拼图

## 核心功能实现

### 地图渲染与交互
- 使用 Konva.js 实现高性能的画布渲染
- 基于 SVG 路径数据绘制中国地图
- 实现拖拽、缩放、点击等交互功能
- 支持触摸设备的手势操作

### 图片处理
- 内置图片压缩功能，优化存储和加载
- 支持多种图片格式和大小
- 实现图片的拖拽填充和位置调整

### 数据管理
- 使用 IndexedDB 存储图片数据
- 使用 localStorage 存储用户设置和视图状态
- 支持数据导入导出，方便备份和迁移

### 导出功能
- 生成高清图片，支持不同比例和格式
- 导出数据文件，包含所有拼图信息
- 实时预览导出效果

## 浏览器兼容性

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持移动设备的触摸操作
- 要求浏览器支持 IndexedDB 和 Canvas API

## 性能优化

- 图片压缩减少存储空间和加载时间
- 按需渲染，只渲染可见区域
- 使用 React.memo 和 useCallback 优化组件渲染
- 批量更新状态，减少重渲染
