# MapLog - 地图与九宫格编辑器

这是一个基于React和Vite的前端应用，提供中国地图和九宫格的可视化编辑功能，支持图片上传、编辑和导出。

## 功能特性

- **中国地图编辑**：在地图上添加、编辑和管理图片，支持拖拽、缩放和旋转操作
- **九宫格编辑**：创建和编辑九宫格布局，添加图片，支持自定义网格大小和间距
- **图片图库**：上传、管理和重用图片，支持拖拽上传和批量操作
- **数据导出**：导出编辑好的地图或九宫格为图片或数据文件
- **数据导入**：导入之前导出的数据，快速恢复之前的编辑状态
- **响应式设计**：适配不同屏幕尺寸，在桌面和移动设备上都能良好显示
- **实时预览**：导出前可预览效果，确保导出结果符合预期

## 技术栈

- React 19.0.0
- TypeScript
- Vite 6.2.0
- Tailwind CSS 4.1.14
- Konva.js (用于画布操作和图形编辑)
- D3.js (用于地理数据处理和地图渲染)
- IndexedDB (本地存储，用于保存图片和编辑状态)
- Lucide React (图标库)
- use-image (图片加载钩子)

## 快速开始

### 前置条件

- Node.js 18.0.0 或更高版本
- npm 或 yarn 包管理器

### 安装步骤

1. 克隆项目到本地
   ```bash
   git clone https://github.com/chyxin071-sys/maplog.git
   cd maplog
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 配置环境变量
   - 复制 `.env.example` 文件并重命名为 `.env.local`
   - 在 `.env.local` 文件中设置 `GEMINI_API_KEY` (如果需要使用AI功能)

4. 启动开发服务器
   ```bash
   npm run dev
   ```

5. 打开浏览器访问 `http://localhost:5173`

## 部署指南

1. 构建生产版本
   ```bash
   npm run build
   ```

2. 构建产物会生成在 `dist` 目录中

3. 将 `dist` 目录部署到任何静态网站托管服务，如：
   - Vercel
   - Netlify
   - GitHub Pages
   - 自己的服务器

## 使用说明

### 基本操作

#### 切换视图
- 点击左侧工具栏的地图图标切换到中国地图视图
- 点击左侧工具栏的九宫格图标切换到九宫格视图

#### 图片上传
- 在图库面板中点击"上传图片"按钮
- 或直接拖拽图片到图库区域
- 支持多张图片同时上传
- 图片会自动转换为Base64格式并存储在本地IndexedDB中

#### 图片管理
- 在图库中点击图片可以查看大图
- 可以删除不需要的图片
- 支持拖拽图片到地图或九宫格中

### 地图编辑

1. **选择省份**：在地图视图中，点击省份区域
2. **添加图片**：从图库中选择图片拖拽到省份上
3. **编辑图片**：
   - 拖拽图片调整位置
   - 使用缩放控制点调整大小
   - 拖动旋转控制点调整角度
4. **删除图片**：选中图片后按Delete键或点击删除按钮
5. **重置省份**：可以将省份恢复到初始状态

### 九宫格编辑

1. **设置网格**：点击工具栏中的网格设置按钮，调整行数、列数和间距
2. **选择格子**：在九宫格视图中，点击网格单元
3. **添加图片**：从图库中选择图片拖拽到网格上
4. **编辑图片**：
   - 拖拽图片调整位置
   - 使用缩放控制点调整大小
   - 拖动旋转控制点调整角度
5. **删除图片**：选中图片后按Delete键或点击删除按钮
6. **重置视图**：点击工具栏中的重置视图按钮，使九宫格完全显示在页面中

### 导出功能

1. **导出为图片**：
   - 点击工具栏中的导出按钮
   - 在导出预览面板中调整设置
   - 点击"保存图片"按钮下载图片

2. **导出为数据**：
   - 点击工具栏中的导出数据按钮
   - 下载包含所有编辑信息和图片数据的JSON文件

### 导入功能

1. **导入数据**：
   - 在图库面板中点击"导入数据"按钮
   - 选择之前导出的JSON数据文件
   - 数据会被导入并应用到当前视图

## 高级功能

### 键盘快捷键
- **Delete/Backspace**：删除选中的图片
- **鼠标滚轮**：缩放视图
- **拖拽空白处**：平移视图

### 触摸操作
- **单指拖拽**：平移视图
- **双指缩放**：缩放视图
- **点击**：选择网格单元或省份

### 响应式设计
- **桌面端**：完整功能，侧边图库
- **移动端**：适配屏幕大小，底部操作栏

## 注意事项

- 图片会被转换为Base64格式存储在本地IndexedDB中，请注意存储空间限制
- 导出的数据文件包含所有编辑信息和图片数据，文件可能会较大
- 建议定期导出数据以备份您的工作
- 浏览器刷新后，编辑状态会保持（存储在IndexedDB中）

## 浏览器兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 不支持IE浏览器
- 推荐使用Chrome浏览器以获得最佳体验

## 故障排除

### 常见问题

1. **图片上传失败**
   - 检查图片格式是否支持（支持JPG、PNG、GIF等常见格式）
   - 检查图片大小是否过大

2. **导出图片模糊**
   - 确保原始图片分辨率足够高
   - 导出时会自动使用3倍像素比，确保清晰度

3. **数据导入失败**
   - 确保导入的是正确格式的JSON文件
   - 确保文件没有损坏

4. **页面卡顿**
   - 减少同时上传的图片数量
   - 清除浏览器缓存

### 性能优化

- 对于大量图片的场景，建议分批上传
- 定期清理不需要的图片，减少存储空间使用
- 对于复杂的编辑操作，可能会有短暂的性能影响
